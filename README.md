# Prize-Premium
Publicly available codes for the paper 'The Prize Premium in Publishing Timelines' for the Journal of Informetrics.

The codes are comprised of five parts:

1.	Original Figures Main Paper (before Revision).ipynb: Python code for generating the main text figures in the original manuscript.
2.	Codes Supplementary Figures and Tables.ipynb: Python code for generating SI tables and figures.
3.	Revision_Reply_Draft.ipynb: Code used for the new analyses in the revision except for the CS-DID experiments.
4.	CS Corrected DID Final.R: R code for the Callaway and Santâ€™Anna difference-in-differences (CS-DID) estimator.
5.	Prize_Premium_Dataset.csv: A copy of our curated dataset (with anonymized Author IDs and Work IDs for protection). We only retain columns that are necessary for replication.
